## Deployment

### Streamlit Cloud

1. Upload files to GitHub repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Deploy from your repository
4. App will be live in 2-3 minutes!

### Local Testing

```bash
pip install -r requirements.txt
streamlit run app.py
```
